# 🍔 Food Ordering Behavior Analysis (Swiggy Dataset)

## 📊 Project Overview
This project focuses on analyzing customer food ordering behavior using a Swiggy-like dataset. The goal is to uncover insights related to customer preferences, decision-making patterns, discount usage, and overall business performance using Power BI.

---

## 🎯 Objectives
- Analyze customer ordering patterns and behavior
- Identify high-value and loyal customer segments
- Evaluate the impact of discounts on revenue and retention
- Understand how decision time affects customer satisfaction
- Build an interactive dashboard for business insights

---

## 🛠️ Tools & Technologies
- Power BI (Dashboard & Visualization)
- DAX (Data Analysis Expressions)
- Power Query (Data Cleaning & Transformation)
- Excel / CSV Dataset

---

## 📂 Dataset Features
The dataset includes:
- User details (age, city)
- Order information (order value, time, meal type)
- Customer behavior (decision time, repeat order)
- Ratings and discount usage

---

## 📈 Key Analysis Performed

### 1. Customer Segmentation
- Segmented users into Premium, Discount Seekers, Regular, and Low Satisfaction groups
- Identified high-value and risky customers

### 2. Order Decision Behavior
- Classified users as Quick, Moderate, and Slow decision-makers
- Analyzed impact on ratings and repeat rate

### 3. Discount Impact Analysis
- Compared orders, ratings, and repeat rates for discount vs non-discount users
- Identified discount dependency patterns

### 4. Cuisine & Demographic Insights
- Evaluated revenue and loyalty across cuisines
- Analyzed age-group behavior and spending patterns

---

## 📊 Dashboard Features
- KPI Cards (Total Orders, Revenue, Avg Rating, Repeat Rate)
- Customer Segmentation (Donut Chart)
- Order Speed Analysis (Bar & Line Charts)
- Discount Impact Analysis
- Cuisine & Age Group Insights
- Interactive Filters (City, Meal Type, Day Type, Age Group)

---

## 🔍 Key Insights
- Quick decision users tend to have higher satisfaction and repeat rates
- Discount-heavy users show lower long-term loyalty
- Certain cuisines drive higher revenue but lower ratings
- Younger users are more price-sensitive, while adults contribute more revenue

---

## 🚀 Business Impact
This dashboard helps businesses:
- Improve customer retention strategies
- Optimize discount policies
- Enhance user experience
- Identify high-performing segments
